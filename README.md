# Simple-Drawing-Language
C# application that draws certain simple shapes and objects

Commands:
  Position pen (moveTo)
  pen draw (drawTo)
  rectangle <width>, <height>
  circle <radius> (2 marks) 
  triangle <side>,<side>,<side>
  pen <colour>
  fill <on/off> 
  
  Loops eg.
  Count = 1 
  Loop for Count
    Circle radius
    Radius+10
    Rectangle width, height
    Width+10
    Height + 10 
    Count+1 
  Endloop

  Ifs eg.
  If <variable>=10 
    Line 1 
    Line 2 
  Endif
  
  Methods eg.
  Method myMethod(parameter list) 
    Line 1 
    Etc 
 Endmethod
